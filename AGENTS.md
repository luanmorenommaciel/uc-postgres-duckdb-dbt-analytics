## Scope

This repository ships a multi-tool-portable agent layer:

- **Architect agents** — planning, trade-off analysis, ADRs. No Bash. One per tech.
- **Developer agents** — implementation, tests, fixes. Has Bash. One per tech.
- **Troubleshooter agents** *(optional, per-tech)* — incident response, root-cause analysis. Tech-specific.
- **Closer agents** — universal pre-merge polish (code review, simplification, documentation). Ground in every tech KB at runtime via the closer-hook protocol.

The agents live under `.claude/agents/` and ground in knowledge bases under `.claude/kb/`. This file is the index every cooperating tool reads first (Claude Code, Cursor, GitHub Copilot, Codex, etc. — see the [AGENTS.md convention](https://agents.md/)).
